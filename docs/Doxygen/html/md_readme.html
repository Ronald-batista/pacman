Como compilar e executar?

<em>OBS: Para compilar é necessario ter o compilador GCC e a biblioteca ncurses</em>

Instalar GCC no linux Debian/Ubuntu

<tt>sudo apt update</tt>

<tt>sudo apt install build-essential</tt>

Instalar biblioteca Ncurses no Debian/Ubuntu

<tt>sudo apt-get install libncurses5-dev libncursesw5-dev</tt>

Compilar

<tt>gcc -o Pacman Pacman.c -lncurses</tt>
